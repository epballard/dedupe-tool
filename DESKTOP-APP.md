# 🖥️ Audit Deduplicator - Desktop App

**Standalone Windows application for deduplicating QB audit exports**

No installation required - just download and run!

## 📥 Download

**[Download AuditDeduplicator.exe](https://github.com/epballard/dedupe-tool/raw/desktop-app/AuditDeduplicator.exe)** (32 MB)

## 🚀 Quick Start

1. **Download** the exe file above
2. **Double-click** to run
3. **Click** "Auto-Load from Downloads"
4. **Select** your Downloads folder
5. **Done!** App automatically processes the 2 newest Excel files

## ✨ Features

✅ Auto-finds your 2 newest QB exports in Downloads  
✅ Automatically deduplicates by FTID  
✅ Shows new items (in latest run but not in previous)  
✅ Exports to Excel with yellow highlighting  
✅ No Python required - fully standalone  
✅ No internet required - works completely offline  
✅ Portable - run from anywhere, no installation  

## 🔒 Privacy & Security

- All processing happens locally on your computer
- No data is sent to the internet
- Your Excel files never leave your machine

## ⚠️ Windows Security Warning

When you first run the app, Windows Defender might show a warning.

**To run:** Click "More info" → Click "Run anyway"
